# Trust OS Operations Dashboard

Operations dashboard for Trust OS verification workflows.

---

## What is Trust OS?

Trust OS is a Decision Verification Platform that helps organizations verify high-impact decisions before execution.

- Decision verification
- Risk evaluation
- Policy enforcement
- Auditability
- Explainability
- API-first integration

---

## Features

- Real-time verification tracking
- Audit timeline with append-only records
- Cryptographic proof display
- Policy compliance metrics

---

## Quick Start

```sh
# Option A — open directly
open index.html

# Option B — serve locally
npx serve .
```

All data is simulated. No backend or API key required.

---

## Dashboard Sections

**Verified Decisions** — every decision evaluated before execution, with recommendation, risk level, and proof hash.

**Audit Timeline** — chronological record of verification requests, source systems, and outcomes.

**Ledger Proof** — cryptographic SHA-256 hash for each decision. Independently verifiable.

**Policy Compliance** — metrics on automatic approvals, manual reviews, and policy rule application.
